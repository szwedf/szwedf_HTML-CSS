働き方・環境

案件は「現場の課題ベース」で選定

ただ人を埋めるだけではなく、現場の課題やあなたのキャリアの方向性を踏まえてアサインします。
「長くいればいるほど、自分の価値が上がっていく現場」を一緒に選んでいきます。

スキルアップ支援

書籍・オンライン講座の補助、資格取得支援、社内勉強会など、インプットの機会を用意しています。
気になっている技術テーマがあれば、ミニ勉強会として企画することも歓迎です。

リモート／ハイブリッド案件もあり

案件によりますが、リモートワークやハイブリッド勤務のプロジェクトも増えています。
働き方の希望も面談時にすり合わせた上で、なるべくマッチした案件をご提案します。

キャリア相談の場を用意

定期的な1on1で、今の現場の状況だけでなく「3〜5年後どうなりたいか」まで一緒に整理します。
インフラ寄りからWeb開発、SRE・クラウドアーキテクトへのステップも視野に入れています。

よくある質問

完全未経験でも応募できますか？

新卒・第二新卒枠については、IT未経験の方も歓迎しています。
ただし、「ある程度は自分でも調べる」「学習時間を確保する」という前提を大切にしています。

どんな技術スタックの案件が多いですか？

インフラは Linux / Windows サーバー運用、JP1、各種クラウド（AWS / GCP）など。
開発は JavaScript / TypeScript、Vue / React、PHP(Laravel) などの Web 系が中心です。

残業や働き方について教えてください。

案件や時期によって変動はありますが、平均残業時間は20時間前後です。
リモート・ハイブリッド勤務の案件もあり、希望を踏まえてアサインしています。

副業や個人開発は可能ですか？

利益相反がない範囲での副業・個人開発は歓迎しています。
技術ブログやポートフォリオサイトの作成なども、むしろ応援していきたいと考えています。

まずはカジュアルに話してみませんか？

「今のスキルで応募していいのか不安」「インフラからWebに寄せていきたい」など、ざっくばらんに相談してください。
応募前提ではないカジュアル面談も実施しています。
